Installation

Installation will be done through NPM system.

NPM

NPM is packaging system with largest software registry. It allows easily creating and sharing packages written in javascript. It is similar to pip for python

NPM is installed as part of nodejs which is available at school computers.

NPM is used through command line.

NPM init

First of all we will use npm init which is interactive walkthrough to set up package/project.

This will create package.json file in repository which contains information for our package. In the end we will not be uploading the package so information does not need to be complete.

NPM install

Then we need to download Eleventy from registry to use it in our project. That is done via npm command install: npm install {package-name}

This would add package to all dependencies which is not what we need this time as Eleventy will be used only in development, for that reason we will use npm install -D @11ty/eleventy

This will edit package.json and add new package-lock.json file and folder node_modules . File package-lock.json contains information what packages are used in this project including dependencies. Folder node_modules contains all installed (downloaded) packages.
